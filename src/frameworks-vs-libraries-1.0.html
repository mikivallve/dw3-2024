Framework vs LibraryPropósito

Biblioteca colección de "cosas" útiles para usar.

Código re-utilizable que atiende necesidades típicas de los desarrolladores:

  • Set de iconos: Font Awesome
  • Funciones de Javascript que aceleran el desarrollo: jQuery
  • Módulos cerrados de funcionalidades típicas: PhotoSwipe
  • UI: Material UI

Frameworkestructura, APIs y componentes para construir.

Herramienta fundacional con arquitectura, manejo de datos y UI para el desarrollo:

  • Features based CSS Framework: Bootstrap
  • Utility based CSS Framework: Tailwind
  • Component based JS Framework: React
  • UI based JS Framework: Svelte
  • Content Management Systems (CMS): Wordpress
