Library Framework de CSSBootstrap

Para equipos es fundamental porque permite:

  • Descentralizar
  • Coordinar trabajo entre devs
  • Identificar quién hace qué cambios
  • Revertir cambios
  • Crear “ramas” para diversificar trabajo
  • Trabajar local y remoto
  • Compartir repositorios

¿Como conecto mi git local a github?

  1. Crear un repositorio en Github (interfaz gráfica)
  2. Si no existe repo local, simplemente se clona el repositorio a tu computadora.
  3. Si ya existe, hay que vincularlo agregando un origen remoto.
git clone https://github.com/mikivallve/dw3-bootstrap-landingpage
git remote add origin {REMOTE-URL} git push origin master
