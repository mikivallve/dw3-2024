Antes de seguir...Básicos de GIT

Github es una plataforma para alojar proyectos basada en git.

Utilizar un servicio como Github nos permite compartir y gestionar proyectos con otros desarrolladores.

Existen otros servicios basados en git como Gitlab, Bitbucket, Gitkraken, etc.

Vincular un repositorio local a github

Primero hay que crear un repositorio en github (vacío).

git remote add origin {REMOTE-URL} git push origin master
