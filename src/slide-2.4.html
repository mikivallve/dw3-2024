IP

La dirección IP es un número que se asocia con cualquier dispositivo que está conectado a una red que utilice el protocolo (Internet Protocol) TCP/IP.

El número asignado a un dispositivo puede ser público o privado, dinámico o fijo. En una red doméstica la IP es privada y suele ser dinámica pero un servidor que aloja sitios web tiene que tener una IP fija y pública.

Cada sitio web en Internet tiene una dirección IP asociada a un servidor. Sin esa dirección, sería imposible localizar tu proyecto en la inmensidad de la web.
