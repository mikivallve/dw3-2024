Dominio

El dominio es el nombre memorable y fácil de recordar que usamos para acceder a un sitio web (ej. google.com).

Su propósito principal es traducir la dirección IP del servidor a un nombre que los humanos puedan entender.

Para que un dominio funcione, debes asociarlo a la dirección IP del servidor donde están alojados los archivos de tu sitio. Es como si a las coordenadas GPS de tu casa le pusieras un nombre: "Mi Casa".
