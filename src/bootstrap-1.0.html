Library Framework de CSSBootstrap

¿Qué es?

  • Toolkit de FrontEnd: ¿ni Framework ni Biblioteca?
  • Un sistema de columnas, clases utilitarias, snippets y componentes

¿Para qué sirve?

  • Simplifica responsive (mobile-first).
  • Aporta funcionalidades comunes.
  • Acelera la implementación (si el diseño lo toma en cuenta).

¿Por qué elegirlo?

  • Ampliamente utilizado.
  • Buena documentación.
  • Sigue estándares de accesibilidad.
  • Es compatible con todos los navegadores.
  • La curva de aprendizaje es baja.
  • Se puede expandir y modificar.
