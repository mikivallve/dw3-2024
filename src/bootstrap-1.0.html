Library Framework de CSSBootstrap

¿Qué es?

  • Toolkit de FrontEnd: ¿ni Framework ni Biblioteca?
  • Un sistema de columnas, clases utilitarias, snippets y componentes

¿Para qué sirve?

  • Simplifica responsive (mobile-first).
  • Aporta funcionalidades comunes.
  • Accelera la implementación (si el diseño lo toma en cuenta).

¿Por qué usarlo?

  • Ampliamente utilizado
  • Buena documentación
