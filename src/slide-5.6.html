Conceptos Básicos de GIT

Github es una plataforma open source para alojar, gestionar archivos online así como colaborar con otros desarrolladores.

Si bien básicamente aloja archivos en la nube es diferente a otros servicios como google drive / dropbox porque está pensado para y por desarrolladores.

  • Utiliza el control de versiones git.
  • Repositorios públicos y privados
  • Gestión de revisión de código
  • Integración continua
  • Github Pages
  • Comunidad

Existen otros servicios basados en git como Gitlab, Bitbucket, Gitkraken, etc.
