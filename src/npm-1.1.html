Node PackageManager

node_modules

Todos los paquetes que se instalan a través de npm ubican sus archivos dentro de node_modules. Esto permite ordenar y mantener el código de terceros.

Los desarrolladores incluyen estas dos carpetas como parte de un grupo de estándares que ayudan a entender y colaborar entre las partes.

dist

Archivos combinados y minificados prontos para distribución / producción.

src / source

Archivos fuente para edición, modularizados, sin complilar ni minificar

lib

En esta carpeta se colocan las dependencias a otras bibliotecas (library).
