CDNContent Delivery Network

Es una red de servidores distribuida geográficamente que acelera la entrega de contenido web al acercarlo a los usuarios finales. Como una red de "mini-almacenes".

Su función es acelerar la entrega de contenido web. Cuando un usuario accede a tu sitio, el CDN le sirve el contenido desde el servidor geográficamente más cercano.

Almacenando contenido cerca del usuario, se reducen los tiempos de carga de página y se obtiene una experiencia web más rápida y eficiente.
